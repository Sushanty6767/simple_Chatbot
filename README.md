🤖 Simple Chatbot using LangChain and API Key

A lightweight AI chatbot built with Python and LangChain that interacts with a Large Language Model (LLM) using an API key. The chatbot accepts user input from the terminal, sends it to the AI model, and returns intelligent responses in real time.

🚀 Features
Interactive command-line chatbot
Built with LangChain
Supports API key authentication
Easy to configure and extend
Continuous conversation loop
Beginner-friendly project structure
🛠️ Technologies Used
Python
LangChain
dotenv
LLM API (OpenAI, Gemini, Mistral, etc.)
